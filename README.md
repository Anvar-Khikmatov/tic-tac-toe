# Tic Tac Toe

A browser-based Tic Tac Toe game built with vanilla JavaScript, HTML, and CSS.

## How to Play

1. Player X always goes first.
2. Click any tile to place your mark.
3. First player to get 3 in a row wins.
4. If all tiles fill up with no winner, it's a tie.
5. Click New Game to reset the board.

## Concepts Practiced

- Factory functions
- IIFE module pattern
- Closures and private variables
- Separation of concerns across gameboard, gameController, and displayController objects
- DOM manipulation
